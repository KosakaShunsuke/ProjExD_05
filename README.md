# アンチヒーロー

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
魔王が城を攻めてきた勇者一行を迎え撃つゲーム。
魔王を矢印キーで動かして、攻めてくる勇者一行を倒したらクリア。

# ゲームの実装
### 共通基本機能
* 背景画像と主要キャラクターの描画

### 担当追加機能
* 魔王が放つビーム：魔王が放つビームのエフェクトに関するクラスで、SHIFTキーを押すことによってビームを放つ。
